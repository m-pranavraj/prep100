# Prep100 – JNV + SSS Coaching Website

> **Your Path to Navodaya Success**

A fully responsive, premium EdTech landing page for **Prep100**, India's online coaching platform for JNV (Navodaya Vidyalaya) Entrance Exams.

## 🚀 Live Demo
Deployed on Netlify / Vercel — see link in repo settings.

## 📁 Project Structure
```
prep100/
├── index.html       # Main website (single-page)
├── style.css        # Full design system
├── script.js        # Interactivity & animations
├── thank-you.html   # Post-registration redirect page
├── logo.png         # Prep100 brand logo
├── hero.png         # Hero section illustration
├── netlify.toml     # Netlify deployment config
├── vercel.json      # Vercel deployment config
└── .gitignore
```

## ✨ Features
- **9 Sections**: Hero, Features, Courses, Stats, Benefits, Testimonials, Register Form, Footer
- **Live Countdown** timer to next batch
- **Animated counters** (10,000+ students, 95% satisfaction)
- **FormSubmit.co** registration form → emails to team
- **Floating WhatsApp** button
- **Sticky CTA** on scroll
- **Fully Responsive** — mobile, tablet, desktop

## 🌐 Deploy

### Netlify
1. Connect this repo in [Netlify](https://netlify.com)
2. Set **Publish directory** to `.` (root)
3. No build command needed

### Vercel
1. Import repo in [Vercel](https://vercel.com)
2. Framework preset: **Other**
3. No build command needed

## 📧 Form Submissions
Registration forms are handled by [FormSubmit.co](https://formsubmit.co) and delivered to the team email.
> On first submission, click the activation email from FormSubmit to enable delivery.
